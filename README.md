# Scenius Digest

Automated biweekly meeting summaries for the Scenius community.

## Overview

This project automatically:
1. Captures meeting transcripts from Fireflies.ai
2. Generates AI-powered summaries
3. Publishes digests to the [Scenius Telegram channel](https://t.me/scenius)

## Setup

### Telegram Bot
- Bot: @sensemaking_bot
- Channel: https://t.me/scenius

### Integrations
- **Fireflies.ai** - Meeting transcription
- **Zapier MCP** - Automation orchestration
- **Telegram Bot API** - Channel publishing

## Meeting Filter

Only meetings with **"scenius"** in the title are included. This captures:
- Scenius Town Hall (biweekly)
- Scenius Fundraising Bi-weekly
- Scenius gem sharing ritual
- Scenius Open-Source Panel Discussion
- Other Scenius-related sessions

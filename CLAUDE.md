# Scenius Digest - Claude Instructions

## Purpose
Generate and publish meeting digests from Scenius community calls to the public Telegram channel @scenius.

## Meeting Filter
Only include meetings with **"scenius"** in the title:
```
keyword:"scenius" scope:title
```

## Digest Format
The digest is public-facing content for sharing with the outside world. Keep it informative and engaging.

```
📋 [Meeting Title] Digest
🗓 [Date] • ⏱ [Duration] min

Summary
[2-3 sentence summary of key discussions and outcomes]

Key Topics
• [Topic 1]
• [Topic 2]
• [Topic 3]
• [Topic 4]

🔗 Full transcript: [Fireflies link]
```

## What to Include
- Meeting title, date, and duration
- High-level summary of discussions
- Key topics and themes covered
- Link to full transcript

## What NOT to Include
- Action items (internal task assignments)
- Specific names in task assignments
- Internal governance details
- Sensitive or private discussions
